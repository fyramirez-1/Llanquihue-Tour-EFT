Llanquihue Tour EFT

👤 Autor del proyecto: Fernanda Yael Ramirez Pavez 
Carrera: Analisis y programacion 
Sede: Online
//Descripción
Llanquihue Tour es un prototipo de sistema desarrollado en Java para una agencia de turismo.
Su objetivo es gestionar clientes y productos turísticos aplicando los principios de la Programación
Orientada a Objetos. Su sistema permite cargar información desde archivos de texto, registrar clientes
y productos, realizar búsquedas, aplicar filtros y administrar la información mediante colecciones.

//Conceptos
Se utilizaron los siguientes conceptos:
- Encapsulamiento
- Herencia
- Composición
- Polimorfismo
- Interfaces
- Sobrecarga de métodos
- Excepciones personalizadas
- Colecciones (ArrayList, HashMap y List)
- Lectura de archivos de texto (.txt)

  //Archivos
  Este proyecto utiliza dos archivos de texto para cargar información:
  -Clientes.txt
  -Productos.txt
  Los cuales son leidos por la clase "LectorArchivos" conviertiendo los datos en objetos.

  //Forma de ejecución
  1. Abrir el proyecto en IntelliJ IDEA.
  2. Ejecutar la clase "Main".
  3. Los resultados se mostrarán en la consola.
